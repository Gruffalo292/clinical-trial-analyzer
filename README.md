# Clinical Trial Protocol Analyzer

An automated tool to analyze clinical trial protocols and generate traffic light indicators for:
- Patient Burden
- DCT Suitability
- Study Budget
- Site Contracting Complexity

## How to Use
1. Upload your protocol document (PDF, DOCX, or TXT)
2. Review the automated analysis
3. Download the comprehensive report
